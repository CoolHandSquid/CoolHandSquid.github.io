File Recovery


lsblk to see all drives/devices
cd into drive/device that file has been lost

mount to see where the data is stored
root@raspberrypi:/media/usbstick# mount
/dev/sdb on /media/usbstick type ext4 (ro,nosuid,nodev,noexec,relatime,data=ordered)

strings to see strings of data in the hex
root@raspberrypi:/media/usbstick# strings /dev/sdb
3d3e483143ff12ec505d026fa13e020b
Damnit! Sorry man I accidentally deleted your files off the USB stick.

xxd /dev/sdb to see in pure hex
grep -a ‘[a-z0-9]\{32\}’ /dev/sdb will grep the hex values for lowercase letter or number, 32 in a row. (isnt regex fun?)
